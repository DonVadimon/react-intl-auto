# Выделение общего кода между visitrs

Теперь у нас есть хорошие jest тесты и мы уврены, что cli и плагин работают одинаково на одинаковых входных данных.

Однако в коде визиторов cli и плагина есть большое количество одинакового кода. Надо вынести его в кор крейт, также, как вынесены например функции is_define_messages_call, is_format_message_call

Подготовь задачу HYBRID_EXTRACT-007F с описанием какой общий код надо вынести и куда. Критерием выполнения является прохождение тестов npm run test:full
